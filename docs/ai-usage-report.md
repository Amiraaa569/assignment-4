## 3. Detailed Use Cases

### Use Case 1 — Fixing Navbar Spacing & Layout
**Prompt:**  
“Fix spacing issues in my navigation bar and remove the extra right-side margin.”

**AI Output (Summary):**
- Suggested using Flexbox (`display: flex`, `gap`)
- Recommended `overflow-x: hidden`
- Proposed splitting the navbar into `.nav-left` and `.nav-right`

**My Modifications:**
- Adjusted gap and padding values
- Improved responsiveness for small screens
- Tuned colors and spacing to match my theme

**What I Learned:**  
Better understanding of Flexbox, alignment, spacing control, and avoiding horizontal scrolling issues.

---

### Use Case 2 — Improving Overall Responsiveness
**Prompt:**  
“My portfolio layout breaks on smaller screens—how do I fix responsiveness?”

**AI Output (Summary):**
- Recommended using CSS Grid

**My Modifications:**

How to build layouts that adapt to different screen sizes more effectively.
